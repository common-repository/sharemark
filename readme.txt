== Description ==

Sharemark lets you integrate ASX market information to your website.

Sharemark is a subscription service that lets you display share prices, charts and ASX announcements on your websites and mobile apps.

Note: Sharemark requires a FREE subscription available [here](http://sharemark.com.au).

== Installation ==

1. Upload the zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register for a FREE account via [Sharemark Website](http://sharemark.com.au/ "Sharemark") and receive a licence key
4. Click on the Sharemark tab in the 'Admin' menu in WordPress and enter your registration details
